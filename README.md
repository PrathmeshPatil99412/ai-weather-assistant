# AI Weather Intelligence System

AI-powered weather intelligence system that integrates real-time weather data with Gemini to generate structured insights, recommendations, and risk analysis using a rule-based validation layer and a clean Streamlit interface.



## Features

- Real-time weather data retrieval using OpenWeather API  
- AI-generated structured analysis using Google Gemini  
- Rule-based validation layer to ensure reliable outputs  
- Clean and minimal Streamlit-based UI  
- Modular and scalable architecture  



## System Architecture

User Input (City)  
↓  
Streamlit UI (app.py)  
↓  
Weather API (weather.py)  
↓  
Rule based Validator (validator.py)  
↓  
Gemini AI (gemini.py)  
↓  
Formatted Output Display  



## Tech Stack

- Python  
- Streamlit  
- Google Generative AI (Gemini)  
- OpenWeather API  
- Rule-based validation
